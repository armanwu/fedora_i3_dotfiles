# fedora_i3_dotfiles
My i3 config for my Fedora 34 based on <a href="https://spins.fedoraproject.org/i3/download/">Fedora i3 Spin</a>.

- distro: Fedora 34
- terminal: URxvt
- launcher: rofi
- wallpaper: <a href="https://pixabay.com/photos/arctic-sea-water-ice-floating-983917/">pixabay</a>
